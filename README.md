# Nick's Umbrel Community App Store

Personal Umbrel community store with apps I maintain.

## Apps

- **Ecobee Dashboard** — Internal dashboard for Ecobee SmartBuildings:
  live status, 5-min runtime historian, alerts, fleet metrics with
  today/week/month/YTD periods, weather widget, animated equipment
  indicators, dark mode, configurable timezone, and cost estimation
  tunable to your LADWP rate.

## Install

In Umbrel: **Settings → App Stores → Add Store** and paste this repo's URL:

```
https://github.com/rvt-nick/ecobee-umbrel-store
```

The apps appear under the **App Store** with the `rvt-nick-` prefix.

On first open, the dashboard's `/setup` page asks for your Ecobee
SmartBuildings API credentials (different from the SB portal login —
obtain by emailing Ecobee SB support).
